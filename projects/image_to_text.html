Image to Text Translation Application

  • Tech Stack: Flutter, Dart, Google ML Kit, Github
  • Github URL: Project Link

Developed a feature-rich translation app using Flutter and Google's Machine Learning Kit, designed to simplify text extraction and translation tasks. The app offers users the flexibility to capture images directly from their device's camera or select images from their gallery. Leveraging the power of Google's ML Kit, it efficiently detects and extracts text from the provided images with high accuracy. This functionality enables users to process written content quickly, regardless of the source, making it highly practical for various scenarios.

Once the text is extracted, the app seamlessly translates it into the user's desired language, ensuring accessibility and convenience. This makes the app a valuable tool for language learners, travelers navigating foreign languages, or anyone needing quick translations for their daily tasks. The combination of Flutter's cross-platform capabilities and Google's advanced machine learning technology ensures a smooth and reliable user experience, showcasing the potential of modern app development and AI integration.
